## Description

ToDoList is a simple to-do list application built using HTML, CSS, and JavaScript. It provides a clean and intuitive interface for managing your tasks and staying organized.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Responsive design for a seamless experience on various devices
- Simple and easy-to-use interface

## Installation

Follow these steps to set up and run the ToDoList app locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/Logeshwari25/ToDoList.git` 

2. Navigate to the project directory:
    
    
    `cd ToDoList` 
    
3. Open the `index.html` file in your preferred web browser.
    

That's it! You now have the ToDoList app up and running on your local machine.

## Usage

- Add tasks using the input field and press Enter or click the "Add" button.
- Edit tasks by clicking on the task text.
- Mark tasks as completed by clicking on the checkbox.
- Delete tasks by clicking on the delete icon.

Feel free to customize and enhance the app according to your needs.
